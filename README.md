# ACNH-FishProject
a program that displays various information for fishing in a specific video game from an SQL database based on certain user input

This program uses an HTML front end with a JavaScript based back end to show the user what fish are available to catch in their specified time and loacation as well as characteristics to look for when searching for a specific fish
The front end contains various input fields for the user to specify their search criteria (fish name, month, time, hemisphere, etc.)
This is sent to the back end as a pull request for the database to return entries that match those criteria
The data is then shown in a table in the front end and can be ordered by any of the above mentioned criteria without reconsulting the back end
There is also a page added for development purposes to easily add new fish to the database

I also made a different version of this program in angular that used a virtual machine as the back end
